# Markdown

- A way to style text on the web
- Similar to regular text with a few non-alphabetic characters like # or *
- Can be used in most places around GitHub

## What is it used for?

1. Formatting words as **bold** or _italic_.
2. Adding images
3. Creating lists

# This is an H1
## This is an H2
### This is an H3
#### This is an H4
